# おわりに

## この本で作ったもの

本書を通じて、以下の技術を統合したモバイルアプリを一通り作りました。

- **Expo SDK 54** でクロスプラットフォーム開発
- **Supabase** でDB設計・RLS・認証・Edge Functions
- **RevenueCat** でサブスクリプション課金
- **EAS Build & Submit** でApp Store公開

「ふくログ」は実際にApp Storeで公開しているアプリです。本書のコードはそのまま動くプロダクションコードをベースにしています。

## 次のステップ

本書でカバーしきれなかった、次に挑戦すると良いテーマをいくつか紹介します。

### プッシュ通知
Expo Notificationsを使えば、月末リマインダーや確定申告の締切通知を実装できます。Supabase Edge Functionsからの送信と組み合わせると強力です。

### アナリティクス
ユーザーがどの画面をよく使っているか、どこで離脱しているかを把握するために、Expo AnalyticsやMixpanelの導入を検討してみてください。

### 多言語対応
`expo-localization` + `i18next` で多言語対応すれば、海外のユーザーにもリーチできます。

### A/Bテスト
課金画面のコピーやレイアウトを変えてコンバージョン率を比較する。RevenueCatのExperiments機能が便利です。

## 個人開発で大切なこと

技術的なスキルと同じくらい大切なのは「出すこと」です。

完璧を目指すよりも、まずMVPをリリースして、ユーザーからのフィードバックを元に改善していく。本書で紹介したExpo + Supabase + RevenueCatの組み合わせは、そのサイクルを最速で回すための技術スタックです。

あなたのアプリがApp Storeに並ぶ日を楽しみにしています。

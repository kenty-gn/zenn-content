副業でアプリ開発をしている方なら、月々かなりの金額をAIツールに払っているはずです。Claude Code、GitHub Copilot、ChatGPT Plus……これらは**きちんと経費になります**。ただし、正しい処理をしないと税務調査で否認されるリスクもあります。

この記事では、副業エンジニアが使うAI開発ツールの経費処理を、具体的な金額と勘定科目で解説します。

:::message alert
この記事は税務の一般的な情報提供を目的としています。個別の状況によって判断が異なる場合があるため、詳細は税理士にご相談ください。
:::

## この記事を読むべき人

- 本業（会社員）＋副業（個人開発、受託開発など）をしている
- Claude Code、GitHub Copilot、ChatGPT Plusなどを月額で課金している
- AIツール代が経費になるか知らなかった
- 確定申告で経費を漏れなく計上したい

## 1. AI開発ツールは経費になるか？

結論から言います。**副業に使っているAIツールの料金は、原則として必要経費にできます。**

根拠は所得税法第37条。「その年分の各種所得の金額の計算上必要経費に算入すべき金額は、別段の定めがあるものを除き、これらの所得の総収入金額に係る売上原価その他当該総収入金額を得るため直接に要した費用の額及びその年における販売費、一般管理費その他これらの所得を生ずべき業務について生じた費用の額とする」——要するに、**収入を得るために使った費用**なら経費になります。

AIツールを副業の開発・執筆・受託業務に使っているなら、その料金は「業務に要した費用」です。

### 2025〜2026年のAI課金事情

2024〜2025年にかけて、エンジニアの開発環境は大きく変わりました。

- Claude CodeがTerminalで動くようになり、実際の開発フローに組み込まれた
- GitHub CopilotがClaude/GPT-4oマルチモデル対応になった
- Cursorがメインエディタとして普及した
- v0やBoltでフロントエンドのプロトタイピングが爆速になった

月に複数のAIツールを課金するのが「普通のエンジニア」になった時代です。これらの費用をきちんと経費処理することは、税負担を合法的に減らす重要な手段です。

### 勘定科目の選び方

AIツールやクラウドサービスのサブスクリプション料は、税法上明確に「これを使え」という規定はありません。実務では以下が一般的です。

- **通信費**: インターネット経由で使うサービス全般に広く使われます。シンプルでわかりやすい
- **支払手数料**: プラットフォームへの手数料的な性質を強調したい場合
- **外注費**: AIツールを使った開発作業を外注と見なす考え方（少数派）

どちらを使っても税務上の問題はありません。**一度決めたら毎年同じ科目を使う**のが重要です（継続性の原則）。

## 3. 按分の考え方

### 副業専用のツールは全額経費でOK

Claude Codeを副業の開発にしか使っていないなら、全額を経費にできます。

問題になるのは、**プライベートと副業の両方で使っているケース**です。

### 按分が必要なケース

```
例: ChatGPT Plusを使っている場合
  - 副業の仕様書作成・コードレビュー: 使用時間の60%
  - プライベートな調べ物・趣味: 使用時間の40%

  経費にできる金額:
    3,000円 × 60% = 1,800円/月
    年間: 1,800円 × 12ヶ月 = 21,600円
```

### 按分の根拠として記録しておくこと

按分率に「正解」はありませんが、合理的な根拠が必要です。

| 記録方法 | 具体例 |
|---------|-------|
| 利用ログ | ChatGPTの履歴から副業関連の会話数をカウント |
| 時間記録 | 副業作業時間 ÷ 総使用時間で算出 |
| 目的別メモ | 各会話の用途（副業 or プライベート）をメモアプリに記録 |

**実務的なアドバイス**: 副業専用のアカウントを作れるサービスは、副業用で別途契約するのがベストです。費用はかかりますが、按分の手間がなくなり、全額経費にできます。

## 4. ドル建て課金の為替処理

Claude Pro、GitHub Copilot、Cursor Proなど、多くのAIツールはドル建てです（2026年3月時点でChatGPT Plusは円建てに変更済み）。

### 原則: 支払い時の為替レートで円換算

```
例: 2025年10月にClaude Pro $20を支払い
  支払い日の為替レート（TTMレート）: 1ドル = 152.3円

  経費計上額: $20 × 152.3円 = 3,046円
```

**TTMレート（仲値）** をクレジットカード利用日の三菱UFJ銀行や三井住友銀行のレートで調べて記録します。

### 実務での簡便法

毎月TTMレートを調べるのは手間がかかります。実務では以下の簡便法も認められています。

```
簡便法: クレジットカード明細の円換算額をそのまま使う

例:
  請求明細: $20.00（USD）→ 3,121円（クレカ換算後）
  この 3,121円 を経費として計上
```

クレジットカード会社の換算レートは実際の支払い額そのものなので、最も実態に即した数字です。**freee や マネーフォワードに明細をインポートすれば自動で処理されます**。

### クレカ明細の保存が最重要

ドル建て課金の領収書処理で最も大切なのは、証拠書類の保存です。

- クレジットカード明細（年間 or 月次）
- サービスのマイページから発行できる請求書（インボイス）
- 利用明細のスクリーンショット

これらを年に一度まとめてフォルダに保存しておくだけでOKです。

## 5. 10万円未満 / 以上の判定

### 1契約ずつで判定する

「年間の合計が10万円を超えたらどうなる？」という質問をよく聞きます。

**答え: AIツールのサブスクは1契約ごとに判定します。**

月額サブスクリプションは、毎月の支払いが独立した費用です。1回あたりの支払いが10万円を超えることはまずないので、減価償却の心配は不要です。

```
例: Claude Pro $20/月 × 12ヶ月 = 年間約36,000円
  → 1回の支払いは約3,000円
  → 毎月「消耗品費 or 通信費 3,000円」として全額経費計上OK

例: 年間プランで一括払い（12ヶ月分を一括）
  → 一括払い額が10万円未満なら全額経費計上OK
  → 例: Cursor Pro 年払い $192 ≒ 28,800円 → 全額OK
```

### 「一括購入したソフトウェア」との違い

JetBrains IDEなどを一括購入した場合（10万円以上）は無形固定資産として減価償却が必要です。ただしほとんどのAI開発ツールはサブスク型なので、この問題は発生しません。

## 6. 具体的な計算シナリオ

### 月3つのAIツールを使う副業エンジニアの場合

```
【プロフィール】
  会社員、副業でアプリ開発（年収入: 48万円）

【AIツール・クラウド費用（月額）】
  Claude Pro:            $20  ≒ 3,000円
  GitHub Copilot:        $10  ≒ 1,500円
  Cursor Pro:            $20  ≒ 3,000円
  Vercel Pro:            $20  ≒ 3,000円
  Supabase Pro:          $25  ≒ 3,750円
  Apple Developer:        ——   1,082円/月相当（年12,980円）
  ──────────────────────────────────
  月合計:                      約15,332円
  年合計:                      約183,984円

【その他の経費】
  MacBook按分（50%）:        100,000円 ※20万円PCを50%按分
  技術書・Udemy等:             30,000円
  通信費按分（30%）:           28,800円
  電気代按分（20%）:           24,000円
  ──────────────────────────────────
  その他経費計:               182,800円

【合計経費】
  183,984円 + 182,800円 = 366,784円

【所得の計算】
  収入:    480,000円
  経費:  - 366,784円
  ──────────────────
  所得:    113,216円 → 20万円以下！
  → 確定申告は不要（ただし住民税の申告は必要）
```

AIツール代だけで年間約18万円が経費になります。これを計上するかしないかで、所得の申告要否が変わってくることもあります。

## 7. 記録の残し方

経費計上のために最低限残すべき証拠書類を整理します。

### 月次でやること（5分でOK）

| 作業 | 方法 |
|------|------|
| 請求書の保存 | 各サービスのマイページ → 請求履歴 → PDFダウンロード |
| クレカ明細の保存 | カード会社アプリのスクリーンショット or CSV出力 |
| 按分記録 | メモアプリに「Claude: 副業80%」など一言残す |

### 各サービスの請求書取得方法

**Claude Pro**
Anthropicのアカウント設定 → Billing → 請求履歴からPDFを取得できます。

**GitHub Copilot**
GitHub Settings → Billing and plans → 請求履歴からダウンロードできます。

**ChatGPT Plus**
ChatGPTの設定 → マイプラン → 請求履歴から取得します（2026年1月以降は円建て・消費税込みの領収書が発行されます）。

**Cursor Pro**
Cursorの Settings → Billing から請求書をダウンロードできます。

### フォルダ構成の例

```
経費記録_2026/
  ├── AI開発ツール/
  │   ├── claude-pro/
  │   │   ├── 2026-01_invoice.pdf
  │   │   └── 2026-02_invoice.pdf
  │   ├── github-copilot/
  │   └── cursor-pro/
  ├── クラウドサービス/
  │   ├── vercel/
  │   └── supabase/
  └── クレカ明細/
      ├── 2026-01_card.pdf
      └── 2026-02_card.pdf
```

年末に「あの領収書どこだっけ」とならないために、毎月保存する習慣をつけるのが一番です。

## 8. まとめ

副業エンジニアがAI開発ツールを経費処理する際のポイントをまとめます。

### チェックリスト

- [ ] **副業専用 or 按分**: 副業だけで使うツールは全額経費。プライベートと混在するなら按分率を決める
- [ ] **勘定科目を統一**: 「通信費」か「支払手数料」、毎年同じ科目を使う
- [ ] **ドル建ては支払い時レートで円換算**: クレカ明細の円建て金額でOK
- [ ] **10万円未満のサブスクは全額経費**: 月額制なら1回あたりの支払いで判定
- [ ] **請求書を毎月保存**: PDF or スクリーンショットをフォルダに保存する

### 経費にできる金額の目安（年間）

| 使用ツール数 | 年間経費目安 |
|------------|------------|
| 1〜2つ | 3〜6万円 |
| 3〜4つ（よく使う副業エンジニア） | 10〜18万円 |
| 5つ以上＋クラウド | 20〜30万円超 |

AI開発ツールへの投資は増える一方です。正しく経費処理して、税負担を適切に減らしましょう。

「AIツールって経費になるの？」と疑問に思っていた方に、この記事が参考になれば幸いです。

# 副業エンジニアの確定申告、20万円ルールを正しく理解する

「副業の収入が20万円以下なら確定申告は不要」

エンジニア界隈でよく聞くこのルール、**半分正解で半分間違い**です。

この記事では、副業をしている（またはこれから始める）エンジニア向けに、20万円ルールの正しい理解と、エンジニアならではの経費の考え方を解説します。

## この記事を読むべき人

- 本業（会社員）＋副業（受託開発、アプリ開発、技術記事執筆など）をしている
- 副業の収入が年間10〜50万円くらい
- 「確定申告って必要なの？」がよくわからない
- 経費に何が入るのか知りたい

## 1. 20万円ルールとは

### 基本の仕組み

**給与所得者（会社員）が、副業で得た所得の合計が年間20万円以下であれば、所得税の確定申告は不要。**

これが20万円ルールの正体です。根拠は所得税法第121条。

ポイントは**「所得」であって「収入」ではない**こと。

```
所得 = 収入 − 必要経費

例：
  副業の収入: 35万円
  必要経費:   18万円（PC、サーバー代、技術書など）
  ─────────────
  所得:       17万円 → 20万円以下なので確定申告は不要
```

つまり、収入が20万円を超えていても、経費を引いた「所得」が20万円以下なら確定申告は不要です。

### よくある誤解

| 誤解 | 正解 |
|------|------|
| 「収入が20万円以下なら不要」 | **所得（収入−経費）が20万円以下なら不要** |
| 「20万円以下なら何もしなくてOK」 | **住民税の申告は別途必要**（後述） |
| 「副業が雑所得なら申告不要」 | **雑所得でも20万円を超えたら申告必要** |
| 「確定申告すると会社にバレる」 | **住民税を「自分で納付」にすればバレにくい** |

### 見落としがちな落とし穴

**医療費控除やふるさと納税（確定申告方式）のために確定申告をする場合、副業所得が20万円以下でも全て申告する必要があります。**

20万円ルール（所得税法第121条）は「確定申告をしない」場合にのみ適用される特例です。一度確定申告書を提出すると決めた時点で、全ての所得を記載する義務が発生します。

つまり：
- ふるさと納税をワンストップ特例ではなく確定申告で処理する人
- 医療費控除を受ける人

は、副業所得が20万円以下でも申告が必要です。これを知らずに副業所得を記載せずに提出すると、後から修正申告を求められる可能性があります。

## 2. 住民税の罠 — 20万円以下でも申告が必要

**ここが最も多くの人が見落とすポイントです。**

所得税の確定申告は20万円以下なら不要ですが、**住民税にはこの免除ルールがありません**。

つまり、副業の所得が**1円でも**あれば、お住まいの市区町村に住民税の申告をする義務があります。

### やるべきこと

| 副業所得 | 所得税（確定申告） | 住民税 |
|---------|-----------------|--------|
| 20万円以下 | 不要 | **市区町村に申告が必要** |
| 20万円超 | **確定申告が必要** | 確定申告をすれば自動的に申告される |

**実務上のアドバイス**: 所得が20万円を超える見込みがあるなら、最初から確定申告をしたほうが楽です。確定申告をすれば住民税の申告も自動で済むので、二度手間になりません。

## 3. 「所得」と「収入」の計算 — エンジニア向け具体例

### シナリオ: 月3〜4万円の副業収入があるエンジニア

```
【収入の部】
クラウドソーシング（受託開発）:    300,000円
技術記事の執筆料（Zenn、note等）:   50,000円
個人アプリの売上:                   80,000円
──────────────────────
収入合計:                         430,000円

【経費の部】
MacBook（按分50%）:               100,000円 ※20万円のPC × 副業使用率50%
クラウド費用（Vercel, Supabase等）:  36,000円 ※月3,000円 × 12ヶ月
ドメイン・サーバー代:                6,000円
技術書・Udemy等:                   25,000円
Apple Developer Program:           12,980円
通信費（按分30%）:                  28,800円 ※月8,000円 × 30% × 12ヶ月
電気代（按分20%）:                  24,000円 ※月10,000円 × 20% × 12ヶ月
──────────────────────
経費合計:                         232,780円

【所得の計算】
430,000円 − 232,780円 = 197,220円
→ 20万円以下！ 確定申告は不要（ただし住民税の申告は必要）
```

収入43万円でも、経費をきちんと計上すれば所得は20万円以下になりえます。

## 4. エンジニアが経費にできるもの一覧

### 堂々と経費にできるもの

| カテゴリ | 具体例 | 勘定科目 |
|---------|--------|---------|
| **PC・ガジェット** | MacBook、モニター、キーボード、マウス | 消耗品費（10万円未満）or 減価償却費 |
| **クラウドサービス** | Vercel、Supabase、AWS、Firebase、Heroku | 通信費 or 支払手数料 |
| **開発ツール** | GitHub（有料プラン）、Figma、JetBrains IDE、GitHub Copilot | 通信費 or 支払手数料 |
| **書籍・教材** | 技術書、Udemy講座、オンラインスクール | 新聞図書費 or 研修費 |
| **ドメイン・サーバー** | お名前.com、Cloudflare、さくら | 通信費 |
| **通信費** | インターネット回線、スマホ代 | 通信費（按分） |
| **Apple/Google登録料** | Apple Developer Program、Google Play | 支払手数料 |
| **決済手数料** | Stripe手数料、App Store手数料 | 支払手数料 |
| **電気代** | 自宅の電気代 | 水道光熱費（按分） |
| **コワーキングスペース** | 作業場所の利用料 | 賃借料 |

**PC・ガジェットの注意点**: 10万円未満なら全額その年の経費（消耗品費）。10万円以上は原則として減価償却が必要です（PCの法定耐用年数は4年）。ただし青色申告であれば30万円未満は一括経費にできる特例があります（少額減価償却資産）。

### 按分（あんぶん）とは

自宅で副業をしている場合、プライベートと業務の両方で使うものは「按分」して経費にします。

```
按分の考え方：
  PC → 副業で使う時間の割合（例: 50%）
  通信費 → 副業で使う時間の割合（例: 30%）
  電気代 → 副業で使う面積or時間の割合（例: 20%）

重要：按分率に「正解」はない。合理的な根拠があればOK。
      ただし、100%経費は税務調査で否認されるリスクが高い。
```

### 経費にできないもの

- スーツ、ビジネスバッグ（IT副業では業務との関連性が薄い）
- 食事代（1人での食事は原則NG。打ち合わせ目的の会食は交際費としてOK）
- 通勤交通費（自宅で副業なら発生しない）
- 副業と関係のない趣味のガジェット

## 5. 雑所得 vs 事業所得 — どちらで申告すべき？

### 基本的な判断基準

副業の規模や継続性によって、「雑所得」と「事業所得」のどちらかに分類されます。

| 判断基準 | 雑所得 | 事業所得 |
|---------|--------|---------|
| 収入規模 | 年300万円以下が目安 | 年300万円超 or 本業級の規模 |
| 継続性 | 単発・不定期 | 反復・継続的 |
| 帳簿の有無 | なくても可 | 帳簿の保存が条件 |
| 青色申告 | 使えない | **使える（最大65万円控除）** |
| 損益通算 | できない | **できる（本業の給与と相殺可能）** |

### エンジニア副業の現実的な判断

- **年収300万円以下、帳簿なし**: → **雑所得**（ほとんどの副業エンジニアはこちら）
- **年収300万円超、帳簿あり**: → 事業所得として申告可能
- **年収300万円以下でも、帳簿をつけて継続的に事業を行っている**: → 事業所得と認められる可能性あり

**2022年の通達改正のポイント**: 帳簿書類の保存がない場合、原則として雑所得に分類されます。ただし、収入が300万円を超え、かつ事業として認められる事実がある場合は事業所得とされることがあります。

### 今の段階では雑所得で十分

副業を始めたばかりなら、まずは雑所得で申告すればOKです。収入が増えて年300万円を超える見込みが出てきたら、開業届を出して事業所得に切り替えることを検討しましょう。

## 6. 確定申告の期限と手順

### 2025年分（令和7年分）のスケジュール

**今まさに申告期間中のもの:**

| 時期 | やること |
|------|---------|
| **2026年2月16日〜3月16日** | **令和7年分（2025年1〜12月の所得）の確定申告期間** |
| **2026年3月16日（月）** | **提出期限** |

### 2026年分（令和8年分）のスケジュール

**今年の副業収入に関するもの:**

| 時期 | やること |
|------|---------|
| 2026年1月〜12月 | 1年分の収入・経費を記録する |
| 2027年2月16日〜3月15日頃 | 確定申告書の提出期間 |

### 令和7年度税制改正の影響

2025年分（令和7年分）から基礎控除が見直されています。

| 合計所得金額 | 旧基礎控除 | 新基礎控除 |
|-------------|-----------|-----------|
| 132万円以下 | 48万円 | **95万円** |
| 132万円超〜655万円以下 | 48万円 | 48万円（変更なし） |

給与所得控除の最低額も55万円→65万円に引き上げられました。いわゆる「103万円の壁」が「160万円の壁」に変わっています。

**ただし、副業の20万円ルール自体に変更はありません。** 副業所得が20万円を超えるかどうかの判定基準はこれまでと同じです。

### 今年（2026年）やるべきこと

**今から経費の記録をつけ始める**のが最も重要です。

年末に「領収書どこだっけ？」とならないように、毎月の収入と経費を記録しておきましょう。

記録すべきもの：
1. **収入**: 日付、金額、どこから（クラウドワークス、Zenn、App Store等）
2. **経費**: 日付、金額、何に使ったか、按分率
3. **領収書・レシート**: 写真に撮ってクラウドに保存

## 7. 会社にバレずに副業するには

確定申告時に住民税の納付方法を**「自分で納付（普通徴収）」**にチェックすれば、副業分の住民税は自宅に届く納付書で自分で払うことになります。これにより、会社の給与天引き（特別徴収）に副業分が上乗せされないため、バレにくくなります。

ただし：
- 市区町村によっては普通徴収に対応していない場合がある
- 確実にバレないという保証はない
- そもそも就業規則で副業が認められているか確認しよう

## 8. まとめ：副業エンジニアがやるべき3つのこと

### 今日やること
1. **毎月の副業収入と経費を記録する仕組みを作る**
2. 領収書・レシートを写真で残す習慣をつける

### 年末にやること
3. 所得（収入−経費）が20万円を超えるか計算する
   - 超える → 確定申告（翌年2〜3月）
   - 超えない → 住民税の申告（市区町村窓口）

### シンプルなフローチャート

```
副業の収入がある
  ├─ 所得 > 20万円 → 確定申告（必須）
  │                   → 住民税は自動で申告される
  │
  └─ 所得 ≤ 20万円 → 確定申告は不要
                      → 住民税の申告は必要（市区町村へ）
```

## おわりに

エンジニアの副業は、経費を正しく計上すれば税負担を大幅に軽減できます。特にクラウドサービスやPC、技術書など、業務に直接関わる出費が多いのがエンジニアの強みです。

「20万円を超えそうかどうか」を常に把握しておくことが、余計な税金を払わないための第一歩です。
